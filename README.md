# DNA Sequence Analyzer

This is a beginner Python project for analyzing DNA sequences.

## Features

- Counts A, T, G, C
- Calculates GC content
- Detects start codon (ATG)
- Detects stop codons (TAA, TAG, TGA)

## How to Run

1. Install Python
2. Run the file

python dna_analyzer.py
